# Home Assistant

An event drivenpersonal home assistant project designed to monitor weather and indoor environment quality. This system integrates voice commands and dashboard widgets to provide real-time information about the home and the outdoors.

## Features

- **Weather Monitoring**
  - View current weather information on the home screen widget.
  - Ask for weather updates via voice commands (e.g., "What is the weather like today?").

- **Room Environment Monitoring**
  - Monitor Temperature and CO2 levels for specific rooms (e.g., Master Bedroom).
  - View environmental stats on the home screen widget.
  - Query specific room conditions via voice commands.
