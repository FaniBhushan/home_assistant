# Features and Requirements

## Use Cases
<!-- List the features or use cases here -->
1. User shall see the Weather information on the home screen widget.
2. User shall be able to ask the assistant to provide the weather information using voice commands.
3. User shall be able to ask the assistant to provide the temperature and CO2 level information of a particular room using voice commands.
4. User shall see the temperature and CO2 level information of a particular room on the home screen widget.

## Functional Requirements
<!-- List the functional requirements here -->
- The system should be able to respond to the prompts like "What is the weather like today?" and "What is the temperature like today?"
- The system should be able to respond to the prompts like "What is the temperature of the master bedroom right now?" and "What is the CO2 level of the master bedroom right now?"
